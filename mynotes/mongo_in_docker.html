  • Get mongo from docker: 
                    docker pull mongo
  • Create run_mongodb.sh and run it: 
                        $ cat run_mongpdb.sh 
                        #!/bin/bash
                        
                        export VOLUME='/home/user/code'
                        
                        docker run -itd \
                            --name mongodb \
                            --publish 4514:5432 \
                            --publish 90:22 \
                            --volume ${VOLUME}:/code \
                            mongo
  • Start mongo shell from docker: 
                        docker exec -it mongodb bash
    Then type mongo
The following information manily from Tutorials Point.
    Database related commands.
  • Show existing databases: 
                        show dbs
  • Show current database: 
                        db
  • Change database (if not exist, create it): 
                        use DATABASE_NAME
    Like use sinnud
  • Drop database: 
                        db.dropDatabase()
    Like drop database "sinnud": 
                        use sinnud
db.dropDatabase()
