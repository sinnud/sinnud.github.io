Show docker images: 

  • sudo docker images
    or 
    sudo docker image ls
Show running containers: 
  • sudo docker container ls
    or 
    sudo docker ps
Show running/died containers: 
  • sudo docker ps
    or 
    sudo docker ps -aq
    just show container ID
Kill running/died containers: 
  • sudo container rm -f [container ID]
    or 
    sudo container rm -f $(ps -aq)
    kill all of them
