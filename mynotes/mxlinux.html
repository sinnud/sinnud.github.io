Install MX Linux, then need to do the following:
  • sshd sudo apt install openssh-server Then check with ssh user@localhost
  • xrdp with sound
    1. Install basic packages: sudo apt install xorgxrdp xrdp
    2. Service operation: sudo service xrdp restart[or stop/start]
    3. For some reason virtualbox popup, need to fix: sudo apt purge virtualbox*
    4. For sound, download: wget http://mxrepo.com/mx/testrepo/pool/test/p/pulseaudio-module-xrdp/pulseaudio-module-xrdp_0.4-0mx19_amd64.deb and install sudo dpkg -i Downloads/pulseaudio-module-xrdp_0.4-0mx19_amd64.deb
  • docker
    1. wget -O docker.gpg https://download.docker.com/linux/debian/gpg
    2. sudo apt-key add docker.gpg
    3. echo "deb [arch=amd64] https://download.docker.com/linux/debian buster stable" | sudo tee /etc/apt/sources.list.d/docker.list
    4. sudo apt update
    5. sudo apt install docker-ce
    6. sudo usermod -a -G docker user
