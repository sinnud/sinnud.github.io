测试函数的编译执行：(Compile and run test RUST function) cargo test --lib -- --nocapture

这里nocapture参数将测试函数里的输出显示出来（不做捕获隐藏）。需要多出--应该是程序缺陷。 (Here use option --nocapture such that the output of test function will be printed out in standard output. Default setting is, the output of test function will be captured. Here extra -- is bug og cargo I think.)

Create new rust project:
  • cargo new proj_name
Compile as debug (cd proj_name):
  • cargo build
Compile for release:
  • cargo build --release
Compile and run:
  • cargo run
Compile documentation (only own code, not libraries used):
  • cargo doc --no-deps
Clean compiling:
  • cargo clean
